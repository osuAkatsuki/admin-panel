# Pre-commit Setup

This repository uses [pre-commit](https://pre-commit.com/) to run code quality checks before commits.

## Installation

1. Install pre-commit:
   ```bash
   pip install pre-commit
   ```
   Or via Homebrew on macOS:
   ```bash
   brew install pre-commit
   ```

2. Install the git hooks:
   ```bash
   pre-commit install
   ```

3. (Optional) Run on all files:
   ```bash
   pre-commit run --all-files
   ```

## What's Included

- **PHP-CS-Fixer**: Automatically fixes PHP code style issues (PSR-12 standard)
- **PHP_CodeSniffer**: Checks PHP code against coding standards
- **PHP Syntax Check**: Validates PHP syntax
- **General hooks**: Trailing whitespace, end-of-file fixes, YAML/JSON validation, etc.

## Configuration Files

- `.pre-commit-config.yaml`: Pre-commit hooks configuration
- `.php-cs-fixer.php`: PHP-CS-Fixer rules (PSR-12 with tabs for indentation)
- `.phpcs.xml`: PHP_CodeSniffer rules (PSR-12 standard)
- `.prettierrc.json`: Prettier configuration (for non-PHP files)

## Running Manually

Run all hooks:
```bash
pre-commit run --all-files
```

Run specific hook:
```bash
pre-commit run php-cs-fixer --all-files
pre-commit run phpcs --all-files
```

## Skipping Hooks

To skip hooks for a commit (not recommended):
```bash
git commit --no-verify
```
